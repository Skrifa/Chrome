Skrifa Help

A small collection of tips so you can do incredible things with Skrifa.

Notes

Creating Notes

Press the plus floating bottom in the lower right corner, a new note will be created in the Notebook you're currently in.

Deleting Notes

Press the trash icon on it, it will ask for your confirmation and then delete it.

Editing Notes

Press the pencil icon on it, it will bring the editor screen where you'll be able to write in your note.

Viewing Notes

If you don't want to edit your note but only want to look at it you can do so in the view screen, to open it just press the eye button on the note, from here you'll be able to export your note, print it and also this is where the math and code rendering will be shown.

Moving a Note to other Notebook

You can easily move Notes between Notebooks just by dragging the note to the Notebook you want on the left bar.

Importing Notes

You can import Skrifa and Markdown notes easily by pressing the import floating button in the lower right corner, they will be added to the notebook you are currently in.

Export as PDF

Exporting as PDF is not native as the HTML, Skrifa or Markdown export so the way to do it is press the Print button and choose save as PDF.

Notebooks

Creating a Notebook

Press the plus squared button in the left bar, it will ask you for your notebook's name and a short description.

Deleting a Notebook

Press the trash icon that appears below it's name and description in the left bar, you'll have to choose a deletion behavior, either moving all the notes it contains to your Inbox or deleting everything inside as well.

Editing a Notebook

You can easily change the name or description of your notebook, just press the pencil icon that appears below it's name and description in the left bar.

The Editor

There are some special tools available for you in the editor:

Custom HTML

The custom HTML tool allows you to add any HTML code you want into your note.

Code Snippet

You can insert a code snippet in your note, it even comes with syntax highlightning for many programming languages!.

MathJax

MathJax is supported for writing math, all you need to do is write two dollar signs $$ and everything you put inside of them will be rendered as math in the view screen of your note. The tool in the editor has some math symbols for you to easily insert.

Add some notes

Your notebook is empty, press the floating plus (+) button to create a note.
